Unity Tools

January 10, 2025

I’ve done some custom Unity tools for different projects and for different companies. Here are some of them:

Timeline Editor

This timeline editor allows you to schedule different types of events in a Unity scene. If you create a Linear timeline, the script will wait for each timeline item to be finished before starting the next one.

If you create a Parallel timeline, the script will play all timeline items at once.

You can also chain multiple timelines together and they will play one after another. At the end, the script lets the website know when all the timelines have finished, prompting a “Go to next lesson” button to appear.

Text Helper

To render text in our projects we were using a combination of Text Mesh Pro (TMP) and a plugin called https://willnode.gitlab.io/texdraw/.

Due to inconsistencies in both plugin’s syntax, and to avoid writing the color hex codes manually there was a need for a tool that could convert the same text into different outputs depending on the plugin being used.

And by using simple color tags like “ Text Here ” we saved time when assigning color to the text and when changing it, as the tool replaces the c1 tag with the first element in the “Colors” list, the c2 tag with the second, and so forth.

Additionally, this also meant we could offload the writing of this intermediate text to people outside the programming team.
