Main contributions to the project

  • Build size optimizations:

    • Replaced heavy 3d text with TMPro;
    • Made sure all textures where POW;
    • Reused a single texture as a color palette that all models can use;
    • Ensured the use of prefabs (and teaching other team members about them);
  • Load time optimizations:
    • Implemented Addressables so that the website only needed to load one scene at the time;
    • Ensured our web-server had the correct Content-Encoding headers, allowing us to not include Decompression Fallback code and leverage WebAssembly streaming.
  • Developed an editor tool to automatically check the formatting of Unity Packages produced by the 3D team.
