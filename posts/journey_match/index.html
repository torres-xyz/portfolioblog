Journey Match

June 10, 2023

Click here to play

At Arkadium, we have recently published our first Unity cross platform game, Journey Match. It runs on browsers, iOS and Android (apps leaving internal beta soon).

Journey Match is a re-imagining of another one of our titles: Mahjongg Dimensions, except instead of being a match 2 game, it’s now match 3!

I worked on this game from concept to release. My contributions were focused mainly on:

  • Code for the core game-play loop,
  • Code for the audio manager,
  • Code and architecture decisions for the rules system, which decides what actions are allowed or which game modes are chosen,
  • Code and architecture decisions for smart asset and object reuse,
  • Shader programming for the overall look of the 3D tiles.
  • Overall development of custom external tools for content creation and management.
