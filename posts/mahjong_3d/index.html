About the project

Developed in one week as a coding challenge I grew overly fond of, this Unity game runs both on Android and on the web through WebGL. Please find the source code in the linked github repo.

There are 3 levels of increasing difficulty, but they never get too hard as you always have 5 minutes to beat each one.

I used Kenney’s art assets for all of the game’s textures.

Music by Yadu Rajiv.
